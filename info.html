Что такое KARDANO?

KARDANO - это инструмент, который поможет в освоении и обучении решетке Кардано

На данный момент приложение является консольным.

В кардано есть два режима:

    standart - подается строка, для которой подбирается квадратик в котором можно выбирать ячейки, чтобы увидеть результат надо нажать пробел
    decode - подается имя файла из которого берется шифротекст. затем появляется квадратик в котором надо бывирать клетки. Чтобы увидеть результат надо нажать пробел.
